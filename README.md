# CV Maker

**A streamlined web application built with React to simplify the resume-building process. Users can input personal details, education history, and professional experience to generate a clean, professionally formatted CV in real-time.**

### 💡 Project Context
This project was developed to master **Complex State Management** in React. Unlike a simple counter or to-do list, a CV maker requires managing nested data structures (like lists of experiences or education) while ensuring the live preview stays perfectly in sync with the input fields.

## Features
* **Live Preview:** Instantly see changes as you type your information.
* **Modular Sections:** Dedicated components for Personal Info, Education, and Practical Experience.
* **Responsive Layout:** A side-by-side or stacked layout optimized for clear data entry and viewing.

## Tech Stack
* **Framework:** React.js
* **Build Tool:** Vite
* **State Management:** React `useState` (handling nested objects/arrays)
* **Styling:** CSS3 (Flexbox/Grid for the resume layout)
